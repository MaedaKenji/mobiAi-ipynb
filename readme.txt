Terdapat 3 folder program yang digunakan dalam mengerjakan tugas akhir ini:

1. datasetCNN
digunakan untuk membuat citra dataset secara manual, program berfungsi untuk menggambar landmark pada iris dan kelopak mata, kemudian di-crop sesuai dengan ukuran mata. Citra yang didapat akan disimpan ke dalam folder pilihan sesuai nama kelas.

2. trainingCNN
digunakan untuk meiatih dan memvalidasi model CNN dari dataset yang telah dibuat menggunakan program sebelumnya.

3. klasifikasiCNN
digunakan untuk menjalankan model klasifikasi CNN yang sudah diintegrasikan dengan perangkat ESP32 yang terhubung melalui Wi-Fi ESP32 itu sendiri.
